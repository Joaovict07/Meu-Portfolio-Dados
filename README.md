## 📂 Estrutura do Repositório

Meus projetos estão organizados por áreas de conhecimento e tecnologia, para facilitar a navegação. Cada pasta de projeto contém seu próprio `README.md` com explicações detalhadas sobre o contexto de negócio, métricas e instruções de uso.

## 🚀 Como Navegar

Para explorar meus trabalhos, recomendo que você navegue até a pasta da área de seu interesse (`power-bi` ou `machine-learning`) e abra a pasta do projeto específico.

Dentro de cada projeto, você encontrará a documentação completa explicando o problema de negócio que foi resolvido, o pipeline de dados construído e os resultados obtidos.
